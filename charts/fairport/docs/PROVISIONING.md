# Provisioning

The Tinkerbell provisioning framework (https://tinkerbell.org/) is included with customizations which can be used to automatically PXE boot and install operating systems on bare metal and/or virtual machines.

## Tools

| Name           | Source                      | Description |
| -              | -                           | -           |
| Tinkerbell     | https://tinkerbell.org/     | Tinkerbell is an open-source  provisioning engine that allows you to automate the deployment of operating systems onto bare metal and virtual machines. |
