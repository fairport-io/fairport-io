# Storage

Openebs (specifically openebs hostpath) is used for persistent storage.  By default distributed storage isn't included so applications should rely on their own replication mechanisms or use external storage providers.

# Tools

| Name           | Source                      | Description |
| -              | -                           | -           |
| Openebs        | https://openebs.io/         | A container native storage solution that provisions persistent storage for containers running in kubernetes clusters. |
