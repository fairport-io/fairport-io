# cert-manager

A fairport tuned and vendored version of [cert-manager](https://cert-manager.io). 

## Summary

[cert-manager](https://cert-manager.io) is a Kubernetes add-on that automates the issuance, renewal, and management of TLS certificates. It integrates natively with your cluster to securely handle these machine identities, ensuring that encrypted connections remain valid and up-to-date without requiring manual intervention.
